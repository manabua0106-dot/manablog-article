# 禁止用語・変換ルール(nmanablog)

## 変換必須
| NG | OK |
|---|---|
| 様々 | さまざま |
| 行なう / 行う | おこなう |
| 〜の方(人に対して) | 〜の人 |
| 2-3(ハイフン) | 2〜3(波ダッシュ) |
| 負荷 | 難易度を上げる・レベルを上げる |
| スキット | 1回5分 等(具体的に) |
| 到達 | 〜できるようになります |
| 見られます | 多いです・よくあります |
| 迷わなくなります | 別の表現に書き換え |
| テーブル | 記事本文で使わない |

## 完全禁止
- 体言止め → 絶対禁止。必ず述語で終わらせる
- 比喩表現全般
- 一人称(私・僕・俺・わたし・当方・筆者)
- 絵文字
- 「でしょう」「全然大丈夫」「〜んです」
- 「後ほど紹介します」「詳しくは後述」「前述の通り」系
- 「可能性」「ケース」「〜の一つ」「状況によって」→ 具体語に

## 冗長表現(原則削除・記事全体で合計3箇所以内)
- 〜という / 〜といった / 〜ように / 〜として
- 〜することができます → 〜できます
- 〜をおこなう → 〜する

## 指示語(完全排除)
これ、それ、あれ、どれ、この、その、あの、どの、これら、それら、ここ、そこ、こうした、そうした、そのまま
※H2直下「ここでは、〜を解説します」のみ許容

## 「こと」の使用制限
- 「〜ことです」で終わらせない
- 1H3内で最大2箇所
- 言い換え：やりたいこと→やりたい内容

## AI感のある表現(禁止)
有効です / 大きな分かれ道 / 成功の鍵 / のが基本です / 基本となります /
刺さる / グッと / 一気に / 道が開ける / 未来が切り拓ける /
効果的です / おすすめです / おすすめします /
不安が軽くなります / 不安が和らぎます → 安心できます /
全級合計 → 英検全体

## 「」の使用ルール
- OK：会話文、固有名詞の初出定義
- NG：単語の強調、心理描写、一般的なフレーズ
- 迷ったら外す

## 記号ルール
- ()は半角。全角（）禁止
- H2に「？」禁止。「！」は使わない
- h2/h3内に句読点を入れない
- 数値カンマ：1,000円。英数字は半角

## 断定表現の緩和
出典なしで「多い」「ほとんど」と断定しない → 「〜の傾向があります」
ソースのない数字・統計データは出さない
