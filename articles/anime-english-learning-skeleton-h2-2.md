# 骨格：H2② 英語学習におすすめのアニメ 中級者向け5選

## 導入文の骨格
- 初心者向け作品に慣れてきた人向け
- 会話スピードが上がり、スラングや文化的な表現も出てくる
- ここでは、リスニングの難易度が上がる中級者向け5作品を紹介する

## ジャンプリンク
- 5作品のH3見出しへのリンク

## H3 id="1b" スポンジ・ボブ(SpongeBob SquarePants)｜会話テンポが速くリスニングの練習になる
- 1文目(結論)：スポンジ・ボブはアメリカで定番のコメディアニメで、初心者向け作品より会話テンポが速い
- ポイント1：会話テンポが速い分、聞き取れないセリフがあっても1話11分と短いので何度でも見返しやすい
- ポイント2：コメディなので笑いながら見られる+1話完結で途中からでも楽しめる
- ポイント3：Netflix・Amazon Prime Videoで視聴可能
- 表：難易度★★☆ / 1話11分 / Netflix・Amazon Prime Video / カジュアルな日常会話
- マーカー候補：「1話11分と短いので、聞き取れなかったセリフがあっても何度でも見返しやすい」

## H3 id="2b" ザ・シンプソンズ(The Simpsons)｜アメリカの日常英語と文化をまとめて学べる
- 1文目(結論)：ザ・シンプソンズは買い物・食事・学校生活などアメリカの日常がリアルに描かれていて、教科書には出ないカジュアルな英語を学べる
- ポイント1：30年以上続く長寿シリーズで話題の幅が広い→英語の語彙もさまざまなジャンルに広がる
- ポイント2：Disney+で全シーズン英語字幕対応
- ポイント3：時事ネタ・風刺が多いので全部わからなくてOK
- 表：難易度★★★ / 1話22分 / Disney+ / アメリカの日常英語・文化表現
- マーカー候補：「買い物・食事・学校生活など、教科書には出ないカジュアルな英語を学べる」

## H3 id="3b" アバター 伝説の少年アン(Avatar: The Last Airbender)｜丁寧なセリフが多く聞き取りやすい
- 1文目(結論)：アバターは中級者向け作品の中ではセリフが丁寧で聞き取りやすく、初心者からのステップアップに向いている
- ポイント1：冒険ストーリー+日常会話→キャラクター同士の掛け合いが多く会話パターンを学べる
- ポイント2：Netflixで英語字幕・英語音声対応
- 表：難易度★★☆ / 1話23分 / Netflix / 丁寧な会話・冒険表現
- マーカー候補：「キャラクター同士の掛け合いが多く、会話のやり取りパターンを学べる」

## H3 id="4b" 僕のヒーローアカデミア(My Hero Academia)｜日本アニメの英語吹替で内容を知っている強みを活かせる
- 1文目(結論)：僕のヒーローアカデミアは日本語で見たことがある人なら、内容を知っている状態で英語版を見られるので学習のハードルが下がる
- ポイント1：Netflixで英語吹替に対応
- ポイント2(注意)：日本アニメの英語吹替は字幕と音声が一致しない場合がある→吹替と字幕が別々に制作されるため
- 表：難易度★★★ / 1話24分 / Netflix(英語吹替) / 日常会話+バトル表現
- alertボックス：「日本アニメの英語吹替は字幕と音声がずれる場合がある」
- マーカー候補：「日本語で見たことがある作品なら、英語に切り替えても内容がわかるのでセリフに集中できる」

## H3 id="5b" リック・アンド・モーティ(Rick and Morty)｜語彙が幅広く上級寄りの作品
- 1文目(結論)：リック・アンド・モーティは語彙の幅が広く会話スピードも速いので、中級者の中でも上級寄りの人向け
- ポイント1：科学用語やスラングも出てくる→わからない単語が多くても気にしすぎない
- ポイント2：Netflixで視聴可能
- ポイント3：初心者向け作品に慣れてから挑戦するのがおすすめ
- 表：難易度★★★ / 1話22分 / Netflix / 幅広い語彙・スラング
- マーカー候補：「わからない単語が多くても気にしすぎず、わかる部分を増やしていく」
