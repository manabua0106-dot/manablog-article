# 骨格：H2① 英語学習におすすめのアニメ 初心者向け5選

## 導入文の骨格
- 選ぶ基準は3つ：(1)1話が短い (2)日常会話が中心でスラングが少ない (3)英語字幕+英語音声が使える
- 全10作品の一覧表を配置(作品名/難易度★/1話の長さ/配信サービス)
- まずは中学英語レベルで聞き取れる初心者向け5作品を紹介する

## ジャンプリンク
- 5作品のH3見出しへのリンク

## H3 id="1a" おさるのジョージ(Curious George)｜セリフがゆっくりで1本目に向いている
- 1文目(結論)：英語学習アニメの1本目に迷ったら、おさるのジョージを選んでおけば間違いない
- ポイント1：セリフがゆっくり+映像だけでストーリーがわかる→聞き取れなくても楽しめる
- ポイント2：YouTube公式チャンネルで無料視聴できる→お金をかけずに始められる
- ポイント3：1話12分→毎日1話見ても負担にならない
- 表：難易度★☆☆ / 1話12分 / YouTube(無料)・Hulu / 日常会話の基本フレーズ
- マーカー候補：「映像だけでストーリーがわかるので、英語が聞き取れなくても楽しめる」

## H3 id="2a" ピーナッツ(The Peanuts)｜短い文が中心で聞き取りやすい
- 1文目(結論)：ピーナッツ(スヌーピー)はセリフが短文中心なので、1文ずつ聞き取る練習に使いやすい
- ポイント1：キャラクターの表情やジェスチャーで内容を推測しやすい
- ポイント2：Apple TV+で視聴可能(The Snoopy Show等)
- 表：難易度★☆☆ / 1話7〜25分 / Apple TV+ / 短い日常フレーズ
- マーカー候補：「キャラクターの表情やジェスチャーで内容を推測しやすいので、聞き取れない部分があっても楽しめる」

## H3 id="3a" ブルーイ(Bluey)｜1話7分で毎日続けやすい
- 1文目(結論)：ブルーイは1話7分なので、通勤中やスキマ時間に毎日1話ずつ見られる
- ポイント1：オーストラリア発の家族アニメ→家族の日常会話がそのまま英語フレーズとして使える
- ポイント2：Disney+で英語字幕・英語音声の両方に対応
- 表：難易度★☆☆ / 1話7分 / Disney+ / 家族の日常会話
- マーカー候補：「家族の日常会話が中心なので、そのまま英語フレーズとして使える表現が多い」

## H3 id="4a" アナと雪の女王(Frozen)｜スラングが少なく英語がきれい
- 1文目(結論)：アナと雪の女王はスラングがほぼなく、はっきり発音されるセリフが多いので初心者の英語学習に向いている
- ポイント1：英語字幕と英語音声が一致→字幕を見ながら聞き取る練習がしやすい
- ポイント2：ストーリーを知っている人が多い→内容理解に頭を使わず英語に集中できる
- ポイント3：映画なので102分あるが、30分ずつ区切って見てOK
- 表：難易度★★☆ / 映画102分 / Disney+ / 丁寧な日常表現
- マーカー候補：「英語字幕と英語音声が一致しているので、字幕を見ながら聞き取る練習がしやすい」

## H3 id="5a" 美女と野獣(Beauty and the Beast)｜セリフがゆっくりで日常の場面が多い
- 1文目(結論)：美女と野獣はセリフのスピードがゆっくりで、食事や挨拶など日常の場面が多い
- ポイント1：実際に使える英語フレーズ(挨拶・お礼・質問)が多い
- ポイント2：アナ雪と同じくディズニー作品で英語がきれい→2作品を交互に見るのもあり
- 表：難易度★★☆ / 映画84分 / Disney+ / 丁寧な日常会話・挨拶
- マーカー候補：「アナ雪と交互に見ると、違うシーンの英語に触れられるので飽きにくい」
