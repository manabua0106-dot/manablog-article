# H3タイプ別 合格テンプレート(nmanablog)

writerはH3を書き始める前に、該当するタイプのテンプレートを確認する。

---

## テンプレートA：理由系H3
H2が「〜の理由」の場合、H3は全て「なぜそう言えるか」に答える。

### 構造
```
1文目：結論 ← ターゲット読者の属性を主語に
2文目：根拠(データ・事実)
パーツ：memo(黄)またはtable
マーカー：読者の行動に直結する一文(理由の説明にはマーカーを当てない)
吹き出し：本文と別角度の補足 or 行動アドバイス
```

### 合格例(KW「英語 やり直し 社会人」)
```
<h3 id="1a">中学英語からやり直せばスピーキングの基礎が身につくから</h3>

英語をやり直したい社会人は、中学英語の文法と単語から始めるだけで日常会話に必要な基礎が身につきます。
中学3年間で習う英語は約1,200語で、日常会話の約8割をカバーしています。

[memo title="中学英語で身につく力"]
<ul>
<li>日常会話で使う基本的な文法(現在形・過去形・疑問文)</li>
<li>買い物や自己紹介で使う基本単語(約1,200語)</li>
<li>簡単な質問に英語で答える力</li>
</ul>
[/memo]

<span class="keiko_yellow">中学英語の復習には1〜2ヶ月あれば十分なので、学生時代に英語が苦手だった人でも短期間でやり直せます。</span>

[say name="manabu" img="https://nmanablog.com/wp-content/uploads/2025/05/IMG_2356-scaled.jpeg"]教材選びに迷ったら「中学英語をもう一度ひとつひとつわかりやすく。」を1冊やり切ってみてください。
<br>
1日15分ずつ進めれば1ヶ月で完了するので、通勤時間でも取り組めます。[/say]
```

## テンプレートB：方法・手順系H3
H2が「〜の進め方」の場合。

### 構造
```
1文目：結論(このステップで何ができるか)
2文目：具体的な手順
パーツ：手順をbox(ol)またはtable
マーカー：安心感(「〜でも大丈夫」「〜なくていい」系)
吹き出し：次の行動を具体的に or サービス訴求
```

---

## テンプレートC：アプリ・教材紹介系H3

### 構造
```
1文目：ターゲットに合う理由
2文目：特徴の説明
パーツ：memo(特徴) or table(基本情報)
マーカー：読者にとってのメリット
吹き出し：使い方のコツ or 無料体験の案内
CTA：【ここに{サービス名}のアフィリエイトリンクを挿入】
kanren：関連記事
```

複数サービスの比較テーブルを含むH3。

## テンプレートE：注意点・対処法系H3

### 構造
```
1文目：結論(何に注意すべきか)
2文目：なぜ注意が必要か
パーツ：alert(赤)で注意点リスト
マーカー：最も重要な注意点 or 安心感
吹き出し：具体的な行動アドバイス
```

---

## 全テンプレート共通の禁止パターン

### 1文目
- NG：状況説明から入る
- OK：結論から入る

### マーカー
- NG：1文目と同じ / パーツの要約
- OK：本文にない新情報 or 行動に直結する一文

### 吹き出し
- NG：本文の言い換え / 「確認してみてください」で終わる
- OK：「何を・どうやって」まで書く
